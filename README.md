# 🎮 Kółko i krzyżyk – gra sieciowa

## 📌 Opis
Projekt gry sieciowej napisanej w Pythonie, umożliwiającej rozgrywkę multiplayer w trybie klient-serwer.

Aplikacja pozwala użytkownikom logować się, grać w czasie rzeczywistym oraz brać udział w trybie turniejowym.

---

## ⚙️ Technologie
- Python
- Socket (komunikacja sieciowa)
- SSL (bezpieczne połączenie)
- MySQL (baza danych użytkowników)

---

## 🚀 Funkcjonalności
- logowanie użytkowników
- gra multiplayer przez sieć
- synchronizacja ruchów w czasie rzeczywistym
- tryb turniejowy
- GUI (interfejs graficzny)

---

## ▶️ Jak uruchomić

1. Uruchom serwer:
```
python serwer.py
```

2. Uruchom klienta:
```
python klient.py
```

---

## 🗂 Struktura projektu
- `serwer.py` – obsługa serwera
- `klient.py` – klient gry
- `gui_tictactoe.py` – interfejs graficzny
- `logowanie.py` – logowanie użytkowników
- `menu.py` – menu gry
